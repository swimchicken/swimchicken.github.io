程式設計課程總整理(一)

2023/6/20 上課紀錄 C c++

此篇主要是複習日誌

0.前言

這篇主要是把我程式語言所學到的c/c++常用指令給寫在blog上,方便我做複習或整理,如果我在內容上出現問題也可以幫我修正,謝謝

註:本次內容皆是採用c語言以及IDE=clion的內容下去實作的,如果對C語言或是CLion有興趣的可以自行看影片學如何安裝及操作

1.變數型別

在c語言中,我們首先必須先了解我們處理的單一資料或者單一變數他會是哪種形式的,以下幾種會是比較常見的形式

short a = 3; //短整數型態
int a = 3;  //整數型態
long h = 3;  //長整數型態
float a = 3.0  //浮點數型態
double a = 3.0  //精確度高浮點數型態
char a = 'c'  //字元型態
char a[20] = "swimchicken"  //字串型態

//其中要特別注意int<=>long以及float<=>double的差別,在於位數的不同

還有其中的無號數型態及布林型態,都是在做特定操作或進階操作才需要用到的

unsigned int a = 3;  //無號數型態
bool a = true  //布林型態

相關的操作會在後續的計算或小型實作中會提到

2.Shell I/O

在我們通常寫C語言中,總是會希望由使用者給測資並且回傳Result的正確解答,那我們就得先去了解每一個變數值所要求的輸出跟輸入格式該怎麼做,且我會在細講我們比較常用的function()

//字串傳入
scanf("%s",a);
//字串傳出
printf("%s",a);

//注意一點由於在scanf中若讀取\0(相當於終止符),他就會停止讀取,例如
scanf("%s",a); //這裡我若輸入"swim chicken"
printf("%s",a);  //讀出來會是"swim"

//若要解決,有一個function是
gets(a);
printf("%s",a); //讀出來會是"swim chicken"

//整數傳入
scanf("%d",&a);
//整數傳出
printf("%d",a);

//浮點數傳入
scanf("%f",&a);
//浮點數傳出
printf("%f",a);

//長整數傳入
scanf("%ld",&a);
//長整數傳出
printf("%ld",a);

3.編成習慣

這個區塊我認為是最重要的,把它放到最後面,我會一一解釋為何最重要以及如何改善,看不懂程式碼也沒關係,當成一個概念就行

  • 空間感

good
int a = 3;

//對比

bed
int a=3;
  • 易讀性
good
int count(int a,int b){
    return (a + b) * b;
}

int main(){
    int a,b;
    printf("%d",count(a,b));
}

//對比

bad
int main(){
    int a,b;
    int plus = a + b;
    int ans = plus * b;
    printf("%d",ans);
}
  • 命名正確
good
int row = 3;
int col = 4;
int array[row][col] = {0};

//對比

bad
int a = 3;
int b = 4;
int array[a][b] = {0};
  • 註解
good
int a = 3;
int b = 4;

int ans = a + b;  //加法

//對比

bad
int a = 3;
int b = 4;

int ans = a + b;
