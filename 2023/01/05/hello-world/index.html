ParticleX主題的測試網頁

2023/1/5 測試 Hexo Node.js ParticleX

這是我第一次嘗試使用hexo的主題來製作一個屬於自己的生活點滴以及程式日記,未來會持續更新進步,也特別感謝作者提供的開源主題以及繪師協助我製作此網頁.

常見指令

創建分頁

$ hexo new "分頁名稱"

啟動本地端伺服器

本地端伺服器位於http://localhost:4000/

$ hexo server

新增靜態文件

靜態文件就是指css js…

$ hexo generate

More info: Generating

更新提交網站

使用hexo本身就有的指令可以直接省下git push的一系列操作

$ hexo deploy
