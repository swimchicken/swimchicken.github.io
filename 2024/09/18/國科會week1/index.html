1.紅綠燈配置

首先依照台灣交通局規定,我們分成紅,黃,綠燈.
其中我們可以依序說明時間配置

  • 黃燈

根據黃燈時間上,依照當前路段的速限或是當前路段的車輛平均車速去做判斷.

  • 紅燈

根據紅燈時間,每個城市的方式不同,其中都是以小時為單位去換算,並且大多數都是以車流方式以一小時為單位去做調整.

float Q = N / T;

Q => 車輛/hr
N => 單位時間內車子總數
T => 觀測時間

其中在台灣中山大學資工研究所的一篇論文中,他們採用的是以高雄市的當月平均車流量為主去做分析.

  • 綠燈

根據綠燈時間,每個路段都會以當前時段分成尖峰時段以及非尖峰時段,例如下圖.

  • 全紅燈

全紅燈是一種針對行人的方式,多半是那種夜市或是行人眾多的鬧區才會有這種全紅燈設計
通常全紅燈的設計時數都不固定,交通部也沒有一定的規定.
